add_module_initializer (di)
Add a module initializer. Module initializers are called as part of the skill_module() call in skill modules. They are called after basic initializations have been run. They can be used for example to initialize, check, and assert dependencies.

Parameters

  • di: dependency initializer, must be a function which takes two arguments. The module m, and a table to which fields should be added (the index metatable). The initializer should not set values directly on the module.
